# RFC 6376 Gap Note

`orinoco-mta` の DKIM 実装は、RSA 鍵を使った受信検証と送信署名の基本フローを対象にしています。

## 現在カバーしている内容

- `rsa-sha256` による DKIM 署名検証
- 複数署名の集約評価
- `simple` / `relaxed` canonicalization
- `h`, `bh`, `b`, `l`, `t`, `x` の主要タグ検証
- 送信時の DKIM 署名付与と鍵ローテーション時の再読込
- 署名生成と検証の相互運用テスト

## 実装範囲外として扱う内容

- Ed25519 を含む RFC 8463 系アルゴリズム
- RSA 以外の公開鍵種別
- DNS 公開鍵ポリシーの細かな運用差異の吸収

## 判断メモ

README の RFC 6376 行は、RSA ベースの DKIM 検証と署名機能を指します。
Ed25519 系は未実装のため、現時点では `対応済み` ではなく `一部対応` のままとします。
